

🔬 IA Lab RLC - Diagnostic Intelligent

Bienvenue dans le laboratoire de simulation RLC assisté par Intelligence Artificielle. Ce projet compare les modèles physiques théoriques avec les prédictions d'un réseau de neurones LSTM.

🛠️ Fonctionnalités

Oscilloscope Interactif : Visualisation en temps réel de la tension $V_s(t)$.

Diagnostic IA : Analyse automatique du régime (critique, apériodique) via Gemini.

Comparaison Keras : Importation de vos données d'entraînement LSTM pour vérifier la fidélité du modèle.

📖 Comment l'utiliser ?

Accédez au lien via le bouton bleu ci-dessus.

Ajustez les curseurs ($R$, $L$, $C$) pour voir la courbe physique évoluer.

Collez votre JSON issu de Python dans la zone "IA" pour superposer les courbes.
